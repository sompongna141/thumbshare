# ThumbSnare — YouTube Thumbnail Studio

## One-line product

Turn a video title and angle into 6 scroll-stopping thumbnail concept packs: image prompts, face expressions, text overlay suggestions, color psychology notes, and A/B test variants.

## Target users

- YouTubers and content creators
- Channel managers and editors
- Media buyers running YouTube ads
- Small agencies serving creator clients

## Problem

Thumbnails are the #1 CTR driver on YouTube, but most creators treat them as an afterthought. They either guess, copy trending formats blindly, or pay editors $50–200 per thumbnail. They need structured concept generation they can hand to an editor or generate directly.

## Why this is worth paying for

- Every YouTuber knows thumbnails matter — proven willingness to pay
- Direct revenue impact: higher CTR → more views → more ad revenue + sponsorships
- Provides structured A/B test variants, not single guesses
- Includes color psychology and face expression guidance — not just "make it blue"
- Outputs are ready-to-use prompts for image generation or editor briefing

## Core workflow

1. User enters video title, topic/angle, target audience, tone, and optional channel context.
2. App generates 6 thumbnail concept packs, each with:
   - Image prompt (Pollinations-ready)
   - Face expression / reaction suggestion
   - Text overlay text + placement
   - Color psychology notes (contrast, emotion, attention)
   - A/B variant hint (what to test against this concept)
3. User previews generated image mocks via Pollinations.
4. User picks favorites, regenerates individual concepts.
5. User exports concept pack (prompt sheet + color notes + A/B plan).

## Pollinations role

- **Text:** concept descriptions, text overlays, color psychology, A/B variant suggestions
- **Image:** thumbnail concept mocks via Pollinations image generation
- Core leverage: structured visual output = high perceived value

## MVP scope

### Must-have

- Video title + angle intake form
- Thumbnail concept generator (6 concepts per brief)
- Image preview for each concept via Pollinations
- Per-concept regeneration
- A/B test pairing suggestions
- Exportable concept pack (print + JSON)
- BYOP Pollinations auth

### Nice-to-have after MVP

- Channel brand kit (fonts, colors, face style)
- Bulk generation for scheduled content calendar
- Click-through rate prediction score
- Thumbnail A/B test results tracker
- Face/expression reference library

## Non-goals for first MVP

- User accounts / cloud persistence
- Direct YouTube API integration
- Payment system
- Video upload / frame extraction

## Success metric for MVP

Creator can go from a video title to 6 usable thumbnail concepts with prompts and A/B plan in under 5 minutes.

## Risks

- Generated image quality varies — prompts need strong scaffolding
- Face expressions are hard to control precisely via text-to-image
- Overwhelming users with 6 concepts if they just want one good thumbnail
- YouTube trends change fast — prompt templates may age

## Product stance

This is not an image generator with a YouTube label.
This is a thumbnail strategy workstation with psychology-backed concept generation.
